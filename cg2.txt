计算机图形学研究内容及应用领域
    计算机图形学是研究如何利用计算机生成处理和显示图形的一门学科
    应用于 工业领域 商业娱乐领域 医疗领域 军事领域

样条概念
    给定一组控制顶点而得到一条曲线，曲线的大致形状由这些点予以控制

Hermite曲线
    给定 4 个矢量 P0 P1 R0 R1，称满足条件的三次多项式曲线 P(t) 为 Hermite 曲线
    P(0) = P0
    P(1) = P1
    P'(0) = R0
    P'(1) = R1

bezier 曲线几何作图

Bezier曲线基函数的表达式和性质，端点位置和切矢量，分割定理
    基函数即 n 次多项式
    性质：正性 权性 对称性 降阶公式 升阶公式 导数 积分 最大值
    端点位置：P0， P1
    端点切矢量：n(P1 - P0)， n(Pn - Pn-1)
    分割定理：给定一点将曲线分成两段，两段曲线仍然可以表示成 Bezier 曲线的形式

曲线连续的条件
    零阶几何连续 GC0 在该点处连续
    一阶几何连续 GC1 条件：GC0，切矢量方向连续
    二阶几何连续 GC2 条件：GC1，副法矢量方向连续，曲率连续

参数形式与隐式形式的优缺点
    参数形式有更大自由度控制曲线曲面形状。易于用矢量和矩阵表示，可以直接几何变换，节约计算量。便于处理斜率无限大。便于从低维扩展到高维。
    隐式表示更容易表示等值线。容易判断求交。容易表示相互位置。

走样的概念和反走样的方法

投影分类
    根据投影中心与投影平面之间的距离分为两类：距离有限为透视投影，距离无限为平行投影。

灭点
    不平行于投影平面的平行线，经过透视后收敛于一点，称为灭点
    主灭点：平行于坐标轴的平行线的灭点，主灭点的个数决定投影平面与坐标轴交点的个数。

齐次坐标的定义和图形学为什么引入齐次坐标
    齐次坐标就是将一个原来是 n 维的向量用一个 n+1 维向量来表示。
    多个变换作用于多个目标，为减少计算，使用变换合成，但各种变换的表示方式不一样，比如平移为矢量加法，旋转和缩放为矩阵乘法，这为变换合成带来困难。使用齐次坐标，使变换的表示法统一，有利于变换合成，也便于硬件实现。

光线跟踪法和全局光照明
    物体表面入射光构成：光源直接照射，其他物体的反射光，透射光
    光线跟踪过程为光线传播逆过程
    终止条件：光线未碰到任何物体 光线碰到了背景 光线经过多次反射折射产生衰减，对视点光强贡献很小 光线的反射和折射次数大于一定值
    优点：实现全局光照明 有消隐功能 有阴影效果 有并行性
    缺点：计算量大 显示慢 视点变化要重新计算

局部光照明
    仅考虑光源照射在物体表面产生的反射光。
    假定物体表面是光滑的且由理想材料构成，只考虑被照明物体的几何对反射光的影响。
    忽略光源的形状和颜色。
    优点：可以模拟不透明物体表面的明暗过度
    缺点：真实感较差，阴影区域边界尖锐

Phong 模型公式及公式中各参数的含义
    I = Ie + Id + Is
      = IaKa + IpKd(L.N) + IpKs(R.V)^n
    Ie 为环境光：
        Ia 环境光亮度
        Ka 物体对环境光的反射系数
    Id 为漫反射光：
        Ip 点光源亮度
        Kd 漫反射系数
        L 光线方向
        N 法向
    Is 镜面反射：
        Ip 点光源亮度
        Ks 镜面反射系数
        R 光线反射方向
        V 视线方向
        n 镜面反射系数

法矢量插值 Phong 着色和 颜色插值 Gourand 着色对比
    Phong：
        计算多边形单位法矢量
        计算多边形顶点单位法矢量
        对多边形顶点法矢量进行插值，获得内部各点法矢量
        利用光照明方程计算多边形内部各点颜色
    Gourand：
        计算多边形单位法矢量
        计算多边形顶点单位法矢量
        利用光照明方程计算顶点颜色
        对多边形顶点颜色进行插值，获得内部各点的颜色

包围盒及层次结构
    包围盒指包围物体的封闭表面，如长方体表面，球面
    层次结构：根据景物分布情况将相距较近的景物组成一组局部场景，相邻各组又组成更大的组，这样整个景物空间组织成树状的层次结构。
    算法：光线先与根节点的包围盒进行求交测试，若无交点，则光线与场景中任何物体无交，否则将光线与子节点的包围盒求交测试，直到求出交点或判断没有交点为止。

纹理的三个坐标空间
    将屏幕像素的四个角点映射到景物空间中可见的物体表面
    将景物坐标空间映射到纹理坐标空间
    将像素所对应的纹理坐标空间中的四边形内的所有纹素的值作加权平均，结果作为物体表面的漫反射系统参与颜色计算。

3D图形绘制流水线
    顶点装配：顶点属性在多个 buffer 中，将不同缓存中的属性提取出来构造成一个完整的顶点数据
    顶点着色：将顶点的位置信息从模型坐标系转换到裁剪坐标系，同时对顶点属性进行一些附加的运算，比如动画等。
    图元装配：将多个顶点组织在一起形成一个图元，比如直线或三角形
    光栅化：确定用哪些像素来表示该图元
    片元着色：根据物体材质，光照明，纹理来计算片元的颜色
    片元测试：裁剪，模板，深度
    混合：将片元颜色和帧缓存中的颜色合并
    帧缓存

三维图形显示流程
    模型坐标系 （模型变换） 世界坐标系 （观察变换） 观察坐标系 （关于视见体的裁剪） （投影） 投影平面 （窗口到视区的变换） 设备坐标系 （显示）
